Les Conditions de la Carte Séjour
Si vous n’êtes pas soumis à la condition d’une demande de visa pour entrer en Tunisie, vous bénéficiez de 3 mois de séjour, à partir de la date d’arrivée. Si vous êtes soumis à la condition d’un visa, commencez par vérifier la durée de séjour qui vous a été accordée.
